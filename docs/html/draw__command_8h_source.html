27 /*-----------------------------------------------------------------------------.
28 | |
29 | Draw Commands |
30 | |
31 | Draw Commands were some of the most overlooked features of PebbleOS. |
32 | Neographics will continue to support them and even expand on them: |
33 | For example, you may have noticed that Draw Commands originally didn't |
34 | circles centered on subpixels, and that having multiple points in a |
35 | circle would lead to weird behavior in lieu of drawing multiple circles |
36 | with different centers. |
37 | |
38 | To facilitate these changes, neographics defines the Draw Command file |
39 | format version 2. Note that previously invalid files may now count as |
40 | valid. Neographics should be backwards-compatible with all valid files |
41 | from Draw Command version 1. |
42 | |
43 | The `flags` byte (offset 1 in commands) now also lets users specify |
44 | specific fallback colors for b/w screens. |
45 | |
46 `-----------------------------------------------------------------------------*/
